2. Context-Aware Paste (Cmd+V)

Paste behavior depends on which section is currently visible (scroll position) or last interacted with:

Scrolled to Todos
Cmd+V →
New todo added to
last expanded project
Item slides in with animation
Scrolled to Snippets
Cmd+V →
New snippet created
title = 1st line
content = full text
Item slides in with animation
Scrolled to Commands
Cmd+V →
New shell command
title = 1st line
command = full text
Item slides in with animation

3. Interaction Model

Action Todos Snippets Commands
Click row Toggle done Copy to clipboard Run command
Cmd+V Paste as todo Paste as snippet Paste as command
Right-click Copy / Delete Copy / Delete Copy cmd / Copy output / Delete
Cmd+V (global) Global hotkey opens Noodl → paste lands in visible section
Feedback Inline animations only — slide in, fade out, flash green. No toasts.

4. New Features Summary

● Unified stream — no tabs, single scrollable view
● Sticky pill bar — click to jump to section, shows counts, highlights active
● Context-aware Cmd+V — paste lands in visible section (todo/snippet/command)
● Smart todo target — paste goes to last expanded project, not always Inbox
● Collapsible sections — each section collapsible, state persists
● Global hotkey — configurable shortcut opens Noodl from anywhere
● Inline animations — no toasts, items animate in/out for feedback
● Snippet paste — Cmd+V on snippets section creates snippet (1st line = title)
● Command paste — Cmd+V on commands section creates shell command

5. What Changes / Gets Removed

✕ Segmented tab picker — replaced by pill bar + unified stream
✕ Toast overlay — replaced by inline animations
✕ Hardcoded Inbox paste target — now uses last expanded project
~ Footer bar — simplified, shows context for current section
~ PopoverView — major refactor to unified stream architecture
