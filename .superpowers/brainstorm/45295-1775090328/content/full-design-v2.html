Complete Feature List

● Unified stream — no tabs, all content in single scroll
● Sticky pill bar — jump to section, shows counts, highlights active
● Cmd+V = todo — always adds to last expanded project (Inbox fallback)
● Unified [+] form — type picker (Todo/Snippet/Command) + title + content fields
● Collapsible sections — Todos/Snippets/Commands collapse, state persists via UserDefaults
● Global hotkey — configurable shortcut to open Noodl from anywhere
● Inline animations — slide in/fade out/flash, no toasts
● Click behaviors — todo: toggle, snippet: copy, command: run
● Context menus — copy + delete on all items, copy output on commands
