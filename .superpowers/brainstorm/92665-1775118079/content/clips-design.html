Interaction Model

→ Captures clipboard text
→ Adds as newest clip (top)
→ If 10+ clips, oldest drops
→ Slide-in animation

→ Copies text to clipboard
→ Flash green feedback
Double click clip
→ Copies text to clipboard
→ Removes from clips list
→ Fade-out animation
✕ button
→ Removes clip (no copy)
Clear all
→ Removes all clips
